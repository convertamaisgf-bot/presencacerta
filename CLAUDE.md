Usar `npx live-server <pasta-do-site>` para preview com hot-reload no browser.

## Regras de construção

- Arquivo único `index.html` com CSS inline no `<style>`
- Sem dependências de build (sem Node, sem bundler)
- Responsivo mobile-first
- Botão WhatsApp fixo na tela
- Seções padrão: Hero · Sobre · Serviços · Depoimentos · Contato · Footer
- Footer sempre com "Desenvolvido por Presença Certa" → link `https://presencacerta.com.br`

## Regra obrigatória: arquivo de prompts

Ao finalizar cada site, criar `prompts.md` na pasta do projeto com todos os prompts Midjourney/DALL-E 3 referentes às imagens do layout — um prompt por slot, numerados e descritos.
